3. git的常用命令

   ```git
   git branch  查看分支
   git switch <branch-name>    git chekcout <branch-name>  切换分支
   git switch -c <branch-name>  创建分支
   git status  查看修改过的文件
   git add .  将代码添加到暂存区
   git commit -m "<>"  将代码提交的版本去
   git push origin <branch-name>  将代码推送的远程仓库
   git merge <branch-name>  合并分支

   ```
